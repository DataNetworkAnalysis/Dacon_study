## 이전 변동 내용 가져오기
```
git pull
```

## 현재 브랜치(현재 위치) 및 해당 레포에 존재하는 모든 브랜치 확인
- 현재 브랜치는 혼자 다른색으로 나타내줌.
```
git branch
```

## 브랜치 이동
```
git checkout 이동하고 싶은 브랜치명
```

## 브랜치 합병
```
git checkout 합병시켜야하는 브랜치명
git merge 합병시킬 내용이 들어있는 브랜치명
git push
```

혹은 github의 해당레포에 가서 main branch로 이동 후 pull & 어쩌고 버튼 클릭
